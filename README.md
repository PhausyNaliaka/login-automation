# QA Login Automation Project

This project demonstrates manual and automated testing of a simple login system.

## Project Overview

The project contains a basic login web application and Selenium automation scripts used to test different login scenarios.

## Technologies Used

* HTML
* CSS
* Python
* Selenium
* Git & GitHub

## Test Scenarios

The automation tests cover the following cases:

* Valid login
* Invalid password
* Empty fields
* Basic login validation

## How to Run the Automation

1. Install Selenium

pip install selenium

2. Run the test script

python tests/test_login.py
